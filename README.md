# ramsey.ai 💰🤖

An AI-powered personal finance companion designed to help users budget, crush debt, and build wealth using automated guidance grounded in Dave Ramsey's 7 Baby Steps philosophy. 

`ramsey.ai` acts as an conversational financial coach, guiding users sequentially through their debt-snowball journeys, emergency fund tracking, and wealth-building strategies.

---

## 🚀 Features

*   **7 Baby Steps Engine:** Dynamically tracks and updates user milestones based on their financial health assessment.
*   **Conversational AI Coach:** Interactive chatbot powered by LLMs, specifically prompt-engineered to offer behavior-first, direct, and practical financial guidance.
*   **Debt Snowball Calculator:** Automatically structures user debts from smallest to largest balance to simulate payoff schedules and create psychological momentum.
*   **Smart Budgeting Assistant:** Analyzes monthly income versus expenses to output a zero-based monthly budget allocation.

---

## 🛠️ Tech Stack

*   **Frontend:** React / Next.js, Tailwind CSS, TypeScript
*   **Backend:** Node.js / Express or Python (FastAPI)
*   **AI/LLM layer:** OpenAI API / Anthropic API via LangChain
*   **Database:** PostgreSQL / MongoDB
